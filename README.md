RC_Car

======
Radio Control System 2+4 channels 433MHz

Проект системы радиоуправления для автомоделей на базе приемного модуля OrangeRx OpenLRS 433
*  433 МГц
*  2 пропорциональных канала
*  4 дискретных канала
*  телеметрия
*  память на 8 моделей

CarTx - Пульт управления
CarRx - Приемник
CaeWin - Приложение для Windows для настройки/управления компонентами системы с компа
